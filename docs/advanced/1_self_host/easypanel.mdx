---
title: Windmill on Easypanel
---

You can deploy Windmill on your VPS using Easypanel.

Easypanel helps you deploy your Windmill instance in just a few clicks using our [official template](https://easypanel.io/docs/templates/windmill).

## Step-by-step guide:

- Install Easypanel on your server using our install script:

  ```bash
  curl -sSL https://get.easypanel.io | sh
  ```

- Create a new project
- Head over to the **Templates** section
- Search and select **Windmill**
- Click on **Create**
- Your app is now deployed and you can click **Open** to access it
